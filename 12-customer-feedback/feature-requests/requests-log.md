## Process

### Logging a Request
1. Check if request already exists in table above
2. If new, create file in `requests/` folder with FR-XXX ID
3. If existing, update the detail file and increment counts

### Prioritization Criteria
- **Reach:** How many customers?
- **Revenue impact:** How much ARR?
- **Strategic fit:** Aligns with strategy?
- **Effort:** How hard to build?
